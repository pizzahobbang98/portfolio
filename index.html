01
AI Smart Home Service MVP

Mother Together

임산부와 배우자의 생활 발화를 AI가 이해하고, 스마트홈 가전 제어와 기록까지 연결하는 AI 기반 홈케어 서비스 MVP입니다.

Demo Repository

Problem

임산부는 컨디션 변화가 잦지만 매번 직접 입력하거나 여러 가전을 하나씩 조작하기 어렵습니다.

Built

음성 발화, AI 의도 분석, ThinQ 공기청정기, Hue 조명, 3D 시뮬레이터, Supabase 기록을 연결했습니다.

Role

서비스 기획, 데이터 분석, 프론트엔드 구현, API 연동, 시연 환경 구성까지 담당했습니다.

Next.js TypeScript Supabase OpenAI API LG ThinQ API FastAPI Python 3D Simulator
Key Point. 단순 화면 프로토타입이 아니라, 사용자의 음성 입력부터 AI 판단, 가전 실행, 조명 반응, 3D 공간 변화, 실행 기록 저장까지 이어지는 시연 가능한 MVP로 구현했습니다.
02
Data Pipeline

Real Estate Transaction Pipeline

공공 부동산 실거래가 데이터를 반복 수집하고 정제하여, 재사용 가능한 분석용 데이터 구조로 만드는 ETL 파이프라인 프로젝트입니다.

Repository

Problem

공공 데이터는 조회는 가능하지만 누적 저장과 정제 구조가 없으면 장기 분석에 재사용하기 어렵습니다.

Built

Open API 수집, 결측치 처리, 타입 변환, 컬럼 정리, 적재 흐름을 Python 기반으로 구성했습니다.

Result

단발성 조회 데이터를 반복 가능한 데이터 자산 형태로 관리할 수 있는 구조를 만들었습니다.

Python ETL Open API Data Cleaning Data Pipeline
Expansion. 이후 지역별 가격 변동을 자연어로 질의하거나, 뉴스·금리 데이터와 연결한 부동산 시장 분석 Agent로 확장할 수 있습니다.
03
VOC Data Analysis

AI Styler Go

여행 중 의류 관리 불편을 커뮤니티 텍스트 데이터로 분석하고, 휴대용 의류 케어 솔루션 컨셉으로 연결한 BX 프로젝트입니다.

Case Study

Problem

여행 중 옷 관리 불편은 존재하지만, 실제 사용자가 어떤 상황에서 불편을 느끼는지 구조화가 필요했습니다.

Built

블라인드, 네이버 블로그, 카페 데이터를 수집하고 키워드 빈도와 맥락을 분석했습니다.

Result

사용자 니즈를 기반으로 AI Styler Go 컨셉을 도출했고, LG전자 DX School BX 프로젝트 최우수상을 받았습니다.

Selenium Web Crawling Text Analysis Wordcloud VOC Analysis
Key Point. 추정이 아니라 실제 사용자의 언어를 기반으로 pain point와 needs를 정리하고, 제품 컨셉으로 연결했습니다.
04
Review Data / UX Insight

CX Review Analysis

커뮤니티 리뷰 데이터를 수집·전처리하고, TF-IDF, K-means, LDA를 활용해 사용자 맥락과 UX 개선 방향을 도출한 프로젝트입니다.

Case Study

Problem

리뷰 데이터에는 광고성 문구, HTML, URL, 이모지, 중복 표현이 섞여 있어 바로 인사이트로 쓰기 어렵습니다.

Built

결측치 제거, 광고성 게시글 필터링, 개인정보 마스킹, 공백 정규화, 불용어 사전 적용을 수행했습니다.

Result

텍스트 품질을 개선한 뒤 군집화와 토픽 모델링으로 고객 맥락을 분석했습니다.

Python Pandas Text Preprocessing TF-IDF K-means LDA
Expansion. 향후 LLM과 결합하면 리뷰 입력 시 불편 유형을 자동 분류하고, UX 개선 가이드를 제안하는 리뷰 해석 Assistant로 확장할 수 있습니다.
Project Focus

How I Build

모든 프로젝트는 문제를 먼저 정의하고, 데이터를 다룬 뒤, 실제 사용 흐름으로 연결하는 방식으로 설계했습니다.

01

Problem First

02

Data Flow

03

API Integration

외부 API와 내부 로직을 연결해 화면에서 실제 동작하는 서비스 경험으로 구현합니다.

04

AI Extension

분석 결과를 LLM, Agent, 자동화 흐름과 연결할 수 있는 방향을 함께 고민합니다.
